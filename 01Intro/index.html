Вступ до JavaScript

JavaScript - мова программування, відпочатку створена для HTML, але на сьогодні - самостійна мова. JS - вільна мова (від ліцензій), але саме назва ліцензована. Для публікаціі стандартів мови вживається термін ECMA-Script (ES) .

Кажучі про JS мається на увазі дві реалізаціі мови: браузерний варіант та NodeJS - збірка самостійної мови. Це мова-інтерпретатор або REPL (Read-Eval-Print Loop).

Для того щоб запустити JS у бразері є декілька можливостей.

  • Включивши код в атрібути полів HTML-документів, які відповідають за оброблення подій, наприклад, атрибут onclick
  • Додавши секцію <script></script>
  • Створивши окремий файл-скрипт та підключивши його тим же тегом <script src="..." ></script> (дозволяється лише одне - або тіло, або файл)
