Document Object Model (DOM)

DOM - концепція, згідно з якою верстка HTML перетворюється у набір об'єктів (ООП). Одже їх ієрархія відповідає розмітці, атрибути тегів перетворюються в атрибути об'єктів. Іншими словами, створюється відповідність між HTML та "деревом" програмних сутностей (об'єктів), причому цей зв'язок двосторонній - зміни у HTML перестворюють "дерево", зміни у "дереві" (прогрмними засобами) впливають на HTML.

Рисами DOM є пошук елементів (об'єктів) та їх колекцій, навігація по "дереву" (вгору-батько, вниз-підлеглі, вбоки-сусіди), а також модифікація дерева програмними засобами.

Пошук елементів: document.getElenentXXXXX та node.querrySelector
